### **Principais pontos do desafio**

Este projeto tem o objetivo de ter um ter um cronometo com as funcionalidade

- Play ✅
- Pause ✅
- Adicionar cinco minutos ao cronometro ✅
- Diminuir cinco minutos ao cronometro ✅
- Dar o play no som quando selecionado ✅
